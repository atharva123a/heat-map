# vis-literacy

Steps to install:

    Backend:

        1. Install Node.js version v18.15.0 based on operating system.
            https://nodejs.dev/en/learn/how-to-install-nodejs/

        2. npm install express
        3. npm install cors

    Frontend:

        1. npm install react react-dom
        2. npm install react-simple-chatbot --save

How to run: 

    Server side: 
        node backend.js

    Client side:    
        npm start
